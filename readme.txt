A widget that automagically populates a menu on your sidebar based on the current page you are on.

== Description ==
Mabe at [Sebo Marketing](http://sebomarketing.com)

This widget automatically populates a menu based on the page you are currently on.


== Installation ==

1. Upload `autonav-widget` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Choose the menu that the autonav widget will use
4. Put the widget in your widget area and configure
